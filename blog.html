در اینجا میخواهیم یک معرفی بسیار کلی از PHP داشته باشیم که بصورت مبتدی متوجه شویم که PHP چیست؟ PHP مخفف PHP Hypertext Preprocessor است که PHP یک اسم خاص در آن می‌باشد.PHP یک زبان برنامه نویسی یا اسکریپت نویسی اپن سورس (Open Source) است. به این معنا که استفاده از PHP کاملا رایگان است و شما هیچ هزینه‌ای برای کار با آن پرداخت نمیکنید.کدهای PHP باید روی یک سرور اجرا شوند. این سرور میتواند یک سرور واقعی باشد و یا سرور موجود در کامپیوتر شما (با استفاده از نرم افزارهایی مثل wamp یا xampp)PHP در بسیاری از سایت های بزرگ جهان استفاده شده و کاربردی جهانی دارد.
